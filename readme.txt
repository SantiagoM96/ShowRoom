Proyecto: E-Commerce. 
Realizado para el curso de Javascript - Coderhouse 2022

Nombre de la página: "SHOWROOM"
Profesor: Santiago Avila

Se trata de un E-Commerce, en la cuál, podrá el usuario seleccionar diversos productos de vestimenta. 
Cada producto seleccionado se sumara a un carrito de compras y se mostrará la suma total de lo
que se haya elegido para sumar. Dentro de aquel, podrá eliminar producto por producto, 
comprar y borrar todo el carrito.

Se utilizan librerías "Sweet Alert" y "Toastify" para informarle al usuario que, efectivamente, 
se realizan cada una de sus interacciones dentro del E-Commerce.
